# Zoo Session Bootstrap

For every task:

1. Read every file in .ai/.
2. Read .ai/01-current-issue.md.
3. Read the GitHub Issue.
4. Read all documents listed in the Documentation Manifest.
5. Check git status.
6. Inspect existing implementation.
7. Continue existing work; never recreate completed work.
8. Implement ONLY the current GitHub issue.
9. Follow documentation if implementation conflicts.
10. Stop after implementation and report:
   - Summary
   - Files created
   - Files modified
   - Manual testing
   - Known limitations
   - Suggested next issue
11. Do not update documentation until implementation is approved.
